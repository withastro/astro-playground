# Contributing

Want to improve the playground? Send a PR!

The project contains two packages:
- `packages/playground`: Astro application with Svelte as frontend.
- `packages/lang-astro`: for Astro syntax highlight 

## After checkout

Once you checked out the project, run `pnpm i` from the root of the repository, and `pnpm dev` to start the dev server
of the playground.

Use `pnpm check` before sending a PR.

## Miscellaneous 

- Once a PR is merged, the changes are deployed in production
